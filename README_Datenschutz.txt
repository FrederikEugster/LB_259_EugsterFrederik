Das Datenset hat keine Namen, nicht von Opfern und nicht von Mitarbeitern der Polizei. Jedoch kann man den Daten zu jedem Vorfall das Datum, die genaue Uhrzeit
und die genaue Location ablesen. Obwohl keine Namen genannt werden, ist das schon eine ziemlich genaue Beschreibung des Vorfalls. Ich denke aber nicht, das es die Datenschutzgesetze
der USA verletzt hat, weil dieses Datenset auch von offizieller Seite von Baltimore Police Department kommt.
